模块

模块可能是一个文件，也可能是一个目录。如果模块是一个目录，Node通常会在这个目录下找一个叫index.js的文件作为模块的入口。

典型的模块是一个包含exports对象属性定义的文件，这些属性可以是任意数据。

下面建立一个currency.js文件作为示范：

js var canadianDollar = 0.91; function roundTwoDecimals(amount) { return Math.round(amount * 100) / 100; } exports.canadianToUS = function(canadian){ return roundTwoDecimals(canadian * canadianDollar); } exports.USToCanadian = function(us){ return roundTwoDecimals(us / canadianDollar); }

这里我们导出了两个函数canadianToUS和USToCanadian，但是roundTwoDecimals和canadianDollar依旧是私有的。
